My Appartment is Hot

A study by Sean Stone

Tech

Two Node.js applications power this project. The first runs on a raspberry pi computer which sits on my countertop and saves temperature data to a Mongo database in the cloud at 20 minute intervals. The indoor temperature is taken with a DS18B20 digital thermometer. The outdoor data is fetched from the Open Weather Map API. The second Node.js application exposes a simple, read-only API to serve the temperature data from the database as JSON, and also serves this page. The SVGs are made with d3.js. Check it out on github.
