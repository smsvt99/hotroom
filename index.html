About

When the season turns, the property management of my appartment disables the air conditioning and we are left with only our windows to cool our rooms. This seems reasonable, particularly for November in northern New England. Yet on November 7th, 2020, the night on which this project began, I recorded the mild outside temperature of 11.1째C (52째F) and the absurd inside temperature of 27.8째C (82째F) with my windows wide open. Although I found this uncomfortable and frustrating, it also struck me as amazing and worth recording, so I stopped trying to sleep and built this instead. You can see the data for those remarkably hot days here.

Tech

Two Node.js applications power this project. The first runs on a raspberry pi computer which sits on my countertop and saves temperature data to a Mongo database in the cloud at 20 minute intervals. The indoor temperature is taken with a DS18B20 digital thermometer. The outdoor data is fetched from the Open Weather Map API. The second Node.js application, hosted on Heroku, exposes a simple, read-only API to serve the temperature data from the database as JSON and this page to consume it. The SVGs are made with d3.js. Check it out on github.
